# 018: 交通データ管理・管理画面

**フェーズ**: Phase 2-A
**優先度**: 高
**依存**: 015（交通関連テーブル作成後）
**参照**: Phase2-Requirements.md セクション 3.1, 8.1, 8.2

---

## 概要

交通サービス（路線バス・デマンド交通・タクシー等）のデータを管理画面から CRUD 操作できるようにする。
Phase 1 の病院管理画面と同様のパターンで実装。

## 技術メモ

- Phase 1 の病院管理画面（`/admin/hospitals`）のパターンを踏襲
- `service_area` は TEXT[] 型 → GIN インデックスで overlaps 検索
- 全操作は `supabaseAdmin`（Service Role Key）経由
- `verifyAdminAuth()` を全 Server Actions で実行
