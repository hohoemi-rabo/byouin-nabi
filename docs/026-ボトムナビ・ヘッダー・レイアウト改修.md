## 技術メモ

- BottomNav は `position: fixed; bottom: 0` で常時表示
- MobileFixedFooter がある場合は BottomNav の上に表示（bottom の padding 調整）
- BottomNav のアクティブ状態は `usePathname()` で判定
- デスクトップではヘッダーナビを拡張、BottomNav は非表示
