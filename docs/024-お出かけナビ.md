# 024: お出かけナビ

**フェーズ**: Phase 2-C
**優先度**: 中
**依存**: 015（facilities テーブル）, 017（Google Maps）, 019（ルート検索）
**参照**: Phase2-Requirements.md セクション 5

---

## 概要

病院以外の外出先（スーパー・役所・銀行等）への交通手段を案内する「お出かけナビ」機能。
