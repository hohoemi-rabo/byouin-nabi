## 技術メモ

- search_logs は INSERT のみ Public（誰でも記録可能）、SELECT は Admin のみ
- ダッシュボードは Server Component を維持（クライアント JS 0B を目標）
- 検索ログ送信は `navigator.sendBeacon()` or `fetch` のバックグラウンド実行
- 統計は SQL の集計クエリで実行（将来的にチャート化を検討）
