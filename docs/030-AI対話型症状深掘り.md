## 概要

基本7問のアンケート回答後、AIが症状に応じた追加質問（2-3問）を生成し、対話形式で情報を深掘りする。これにより、より精度の高い緊急度判定・診療科推奨を実現する。

**シニアファースト**: 基本7問はそのまま維持（簡単・ボタン式）。追加質問はオプショナルで、「もっと詳しく調べる」ボタンを押した場合のみ発動。

## 技術メモ

- Gemini API の1回目: 緊急度判定（既存）、2回目: 追加質問生成、3回目: 再判定
- API コスト: 1ユーザーあたり最大3回のAPI呼び出し（gemini-3.1-flash-lite-preview なら低コスト）
- 追加質問は Gemini に JSON で返させる（`responseMimeType: "application/json"`）
- 基本7問の回答は変更しない（既存フローを壊さない）
- 「もっと詳しく調べる」はオプション（押さなくても初回判定で十分使える）
