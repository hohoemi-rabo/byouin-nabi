## 概要

## 技術メモ

- スコアリングはクライアントサイドで実行（API レスポンス後に計算）
- 距離計算: Google Maps Distance Matrix API or 市町村一致で簡易判定
- 営業中判定: 現在の曜日・時刻と `hospital_schedules` を比較
- Phase 1 の既存コンポーネントは可能な限り維持（破壊的変更を最小化）
