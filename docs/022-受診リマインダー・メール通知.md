# 022: 受診リマインダー・メール通知

**フェーズ**: Phase 2-B
**優先度**: 中
**依存**: 015（visit_reminders テーブル）, 020（Supabase Auth）
**参照**: Phase2-Requirements.md セクション 6.5, 13.3

---

## 概要

定期通院の次回受診日をリマインダーとして設定し、前日にメール通知を送信する機能。

## 技術メモ

- Resend 月3,000通まで無料プラン
- 通知は前日18:00（JST）にバッチ送信
- `notified_at` で重複送信を防止
- 定期通院: `next_visit_date` + `interval_days` で自動更新
